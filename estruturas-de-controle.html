Operadores e Estruturas de Controle

Domine o fluxo de controle em Go

Operadores Aritméticos

Operador Descrição Exemplo
+ Adição 5 + 3 = 8
- Subtração 5 - 3 = 2
* Multiplicação 5 * 3 = 15
/ Divisão 10 / 3 = 3
% Módulo 10 % 3 = 1 
package main

import "fmt"

func main() {
    a, b := 10, 3
    
    fmt.Println("Adição:", a + b)        // 13
    fmt.Println("Subtração:", a - b)     // 7
    fmt.Println("Multiplicação:", a * b) // 30
    fmt.Println("Divisão:", a / b)       // 3
    fmt.Println("Módulo:", a % b)        // 1
}

Estrutura If/Else

A estrutura if permite executar código condicionalmente. Em Go, NÃO use parênteses ao redor da condição.

If Básico

package main

import "fmt"

func main() {
    idade := 20
    
    // If simples
    if idade >= 18 {
        fmt.Println("Maior de idade")
    }
    
    // If-else
    if idade >= 18 {
        fmt.Println("Pode votar")
    } else {
        fmt.Println("Não pode votar")
    }
    
    // If-else if-else
    nota := 85
    
    if nota >= 90 {
        fmt.Println("Nota A")
    } else if nota >= 80 {
        fmt.Println("Nota B")
    } else if nota >= 70 {
        fmt.Println("Nota C")
    } else {
        fmt.Println("Nota D")
    }
}

If com Statement Inicial

Go permite declarar uma variável no próprio if. Essa variável só existe no escopo do if/else.

// Declaração de variável dentro do if
if x := 10; x > 5 {
    fmt.Println("x é maior que 5")
}
// x não está disponível aqui!

// Padrão comum: checando erros
if err := fazerAlgo(); err != nil {
    fmt.Println("Erro:", err)
}
⚠️ Importante: Em Go, NÃO use parênteses ao redor da condição do if. Isso é um erro de sintaxe.

Estrutura Switch

O switch em Go é mais poderoso que em muitas outras linguagens e NÃO tem fall-through automático.

Switch Básico

dia := "segunda"

switch dia {
case "segunda":
    fmt.Println("Início da semana")
case "terça", "quarta", "quinta":
    fmt.Println("Meio da semana")
case "sexta":
    fmt.Println("Sextou!")
case "sábado", "domingo":
    fmt.Println("Final de semana")
default:
    fmt.Println("Dia inválido")
}
💡 Diferença importante: Em Go, o switch NÃO tem "fall through" automático. Você não precisa usar break ao final de cada case!

Switch com Condições

nota := 85

// Switch sem expressão (como if-else)
switch {
case nota >= 90:
    fmt.Println("Excelente!")
case nota >= 80:
    fmt.Println("Muito bom!")
case nota >= 70:
    fmt.Println("Bom")
case nota >= 60:
    fmt.Println("Suficiente")
default:
    fmt.Println("Insuficiente")
}

Loops (For)

Go tem apenas uma estrutura de loop: for. Mas ela é extremamente versátil.

For Clássico

// For com inicialização, condição e pós-iteração
for i := 0; i < 5; i++ {
    fmt.Println(i)
}

// Contagem regressiva
for i := 10; i >= 0; i-- {
    fmt.Println(i)
}

For como While

// For com apenas condição
contador := 0
for contador < 5 {
    fmt.Println(contador)
    contador++
}

Loop Infinito

contador := 0

for {
    contador++
    fmt.Println(contador)
    
    if contador == 5 {
        break  // Sai do loop
    }
}

For com Range

// Range em slice
numeros := []int{10, 20, 30, 40, 50}

for indice, valor := range numeros {
    fmt.Printf("Índice: %d, Valor: %d\n", indice, valor)
}

// Ignorando o índice com _
for _, valor := range numeros {
    fmt.Println(valor)
}

// Range em map
idades := map[string]int{
    "João":  25,
    "Maria": 30,
}

for nome, idade := range idades {
    fmt.Printf("%s tem %d anos\n", nome, idade)
}

Break e Continue

// Break - sai do loop
for i := 0; i < 10; i++ {
    if i == 5 {
        break  // Para o loop quando i = 5
    }
    fmt.Println(i)
}

// Continue - pula para próxima iteração
for i := 0; i < 10; i++ {
    if i%2 == 0 {
        continue  // Pula números pares
    }
    fmt.Println(i)  // Imprime apenas ímpares
}

Exemplo Prático: Calculadora

package main

import "fmt"

func calculadora(a, b float64, operacao string) (float64, error) {
    switch operacao {
    case "+":
        return a + b, nil
    case "-":
        return a - b, nil
    case "*":
        return a * b, nil
    case "/":
        if b == 0 {
            return 0, fmt.Errorf("divisão por zero")
        }
        return a / b, nil
    default:
        return 0, fmt.Errorf("operação inválida")
    }
}

func main() {
    resultado, err := calculadora(10, 5, "+")
    if err != nil {
        fmt.Println("Erro:", err)
        return
    }
    fmt.Println("Resultado:", resultado)
}

Boas Práticas

✅ If/Else

  • Não use parênteses ao redor da condição
  • Use if com statement inicial para limitar escopo
  • Prefira early return ao invés de else desnecessário
  • Mantenha condições simples e legíveis

✅ Switch

  • Use switch sem expressão para substituir if-else encadeados
  • Aproveite que não precisa de break
  • Use múltiplos valores em um case quando aplicável
  • Coloque o caso mais comum primeiro

✅ Loops

  • Use range sempre que possível para iterar coleções
  • Use _ para ignorar valores desnecessários no range
  • Evite modificar a variável de iteração dentro do loop
  • Cuidado com referências em range (o valor é copiado)

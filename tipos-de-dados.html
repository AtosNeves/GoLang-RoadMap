Variáveis em Go

Em Go, existem diversas formas de declarar variáveis. A linguagem é estaticamente tipada, o que significa que o tipo de uma variável é definido em tempo de compilação.

1. Declaração com var

A forma mais explícita de declarar variáveis:

declaracao_var.go 
var nome string = "João"
var idade int = 25
var ativo bool = true

// Declaração sem inicialização (valor zero)
var contador int  // 0
var texto string     // ""
var vazio bool       // false

2. Declaração com Inferência de Tipo

Go pode inferir o tipo automaticamente:

inferencia.go 
var nome = "Maria"      // string inferido
var idade = 30             // int inferido
var salario = 5000.50      // float64 inferido

3. Short Declaration (Declaração Curta)

A forma mais comum e concisa, usando o operador :=

short_declaration.go 
// Só pode ser usado dentro de funções
nome := "Carlos"
idade := 28
ativo := true

// Múltiplas variáveis
x, y := 10, 20
nome, idade := "Ana", 22
⚠️ Atenção: O operador := só pode ser usado dentro de funções. Para variáveis em nível de pacote, use var.

4. Declaração em Bloco

bloco.go 
var (
    nome     string = "Pedro"
    idade    int    = 35
    altura   float64 = 1.75
    casado   bool   = true
)

Valores Zero

Quando uma variável é declarada sem valor inicial, ela recebe um "valor zero" baseado em seu tipo:

Tipo Valor Zero Descrição
int, int8, int16, int32, int64 0 Zero numérico
float32, float64 0.0 Zero decimal
string "" String vazia
bool false Falso
Ponteiros, slices, maps, channels nil Referência nula

Go possui diversos tipos de dados primitivos e compostos. Vamos explorar os principais:

Tipos Numéricos Inteiros

inteiros.go 
var idade int = 25
var populacao int64 = 7800000000
var byte1 uint8 = 255  // byte é alias para uint8
var unicode rune = 'A'    // rune é alias para int32

Tipos Numéricos de Ponto Flutuante

float.go 
var preco float32 = 19.99      // 32 bits, precisão ~6 casas decimais
var pi float64 = 3.14159265359 // 64 bits, precisão ~15 casas decimais

// Notação científica
var grande float64 = 1.5e10  // 15000000000
var pequeno float64 = 2.5e-3 // 0.0025
💡 Dica: Use float64 como padrão para números decimais, pois é mais preciso e é o tipo padrão quando você usa inferência de tipo.

Constantes

Constantes são valores que não podem ser alterados durante a execução do programa. São declaradas usando a palavra-chave const.

Declaração de Constantes

constantes.go 
// Constantes simples
const Pi = 3.14159
const Nome = "Sistema"
const Versao = 1

// Constantes tipadas
const Idade int = 25
const Ativo bool = true

// Bloco de constantes
const (
    StatusOk       = 200
    StatusNotFound = 404
    StatusError    = 500
)

Constantes com iota

O iota é um identificador especial usado para criar sequências de constantes enumeradas:

iota.go 
// iota começa em 0 e incrementa
const (
    Segunda = iota  // 0
    Terca           // 1
    Quarta          // 2
    Quinta          // 3
    Sexta           // 4
    Sabado          // 5
    Domingo         // 6
)

// Começando de 1
const (
    Janeiro = iota + 1  // 1
    Fevereiro            // 2
    Marco                // 3
    Abril                // 4
)

// Potências de 2 (tamanhos de memória)
const (
    _  = iota  // ignora o primeiro valor usando _
    KB = 1 << (10 * iota)  // 1 << 10 = 1024
    MB                        // 1 << 20 = 1048576
    GB                        // 1 << 30 = 1073741824
    TB                        // 1 << 40 = 1099511627776
)

// Usando múltiplos valores por linha
const (
    a, b = iota, iota + 10  // 0, 10
    c, d                       // 1, 11
    e, f                       // 2, 12
)
✅ Boas Práticas:
  • Use constantes para valores que nunca mudam
  • Nomeie constantes com PascalCase quando exportadas (públicas)
  • Use iota para criar enumerações
  • Agrupe constantes relacionadas em blocos

Conversão de Tipos

Go não faz conversões automáticas de tipos. Você deve converter explicitamente:

Conversões Numéricas

conversao_numerica.go 
var x int = 42
var y float64 = float64(x)  // int para float64
var z int = int(y)        // float64 para int (perde a parte decimal)

// Conversões entre tipos inteiros
var a int32 = 100
var b int64 = int64(a)
var c uint = uint(a)

// Cuidado com overflow!
var grande int32 = 1000000
var pequeno int8 = int8(grande)  // overflow - valor truncado!
⚠️ Cuidado com Overflow: Ao converter de um tipo maior para menor, você pode perder dados se o valor exceder o limite do tipo destino.

Conversões entre String e Números

string_conversao.go 
package main

import (
    "fmt"
    "strconv"
)

func main() {
    // String para Int
    str := "123"
    num, err := strconv.Atoi(str)  // retorna int e error
    if err != nil {
        fmt.Println("Erro na conversão:", err)
    }
    fmt.Println(num)  // 123
    
    // Int para String
    numero := 456
    texto := strconv.Itoa(numero)
    fmt.Println(texto)  // "456"
    
    // String para Float
    strFloat := "3.14"
    f, err := strconv.ParseFloat(strFloat, 64)  // 64 bits
    fmt.Println(f)  // 3.14
    
    // Float para String
    pi := 3.14159
    strPi := strconv.FormatFloat(pi, 'f', 2, 64)  // 'f' formato, 2 casas decimais
    fmt.Println(strPi)  // "3.14"
    
    // String para Bool
    strBool := "true"
    b, err := strconv.ParseBool(strBool)
    fmt.Println(b)  // true
    
    // Bool para String
    ativo := true
    strAtivo := strconv.FormatBool(ativo)
    fmt.Println(strAtivo)  // "true"
}

Conversões entre Rune, Byte e String

rune_byte.go 
// String para slice de bytes
str := "Olá"
bytes := []byte(str)
fmt.Println(bytes)  // [79 108 195 161]

// Slice de bytes para string
novaStr := string(bytes)
fmt.Println(novaStr)  // "Olá"

// String para slice de runes
runes := []rune(str)
fmt.Println(runes)  // [79 108 225] (caracteres Unicode)

// Rune para string
r := 'A'
s := string(r)
fmt.Println(s)  // "A"

// Int para rune (código Unicode)
codigo := 65
caractere := rune(codigo)
fmt.Println(string(caractere))  // "A"

Exemplo Prático Completo

Vamos criar um programa que demonstra o uso de variáveis, tipos e constantes:

exemplo_completo.go 
package main

import (
    "fmt"
    "math"
)

// Constantes em nível de pacote
const (
    EmpresaNome = "TechCorp"
    Versao      = "1.0.0"
    Pi          = 3.14159265359
)

// Enumeração com iota
const (
    Admin = iota  // 0
    Editor         // 1
    Viewer         // 2
)

// Variáveis em nível de pacote
var (
    totalUsuarios int
    sistemaAtivo  bool = true
)

func main() {
    // Declarações de variáveis
    var nome string = "João Silva"
    idade := 25
    altura := 1.75
    
    // Calculando IMC
    peso := 70.5
    imc := peso / math.Pow(altura, 2)
    
    // Formatando saída
    fmt.Printf("Sistema: %s v%s\n", EmpresaNome, Versao)
    fmt.Printf("Nome: %s\n", nome)
    fmt.Printf("Idade: %d anos\n", idade)
    fmt.Printf("Altura: %.2f m\n", altura)
    fmt.Printf("Peso: %.1f kg\n", peso)
    fmt.Printf("IMC: %.2f\n", imc)
    
    // Usando constantes
    raio := 5.0
    area := Pi * raio * raio
    fmt.Printf("Área do círculo: %.2f\n", area)
    
    // Conversões de tipos
    idadeFloat := float64(idade)
    anosRestantes := 100.0 - idadeFloat
    fmt.Printf("Anos restantes até 100: %.0f\n", anosRestantes)
    
    // Verificando permissão
    permissao := Admin
    if permissao == Admin {
        fmt.Println("Usuário com permissão de administrador")
    }
    
    // Múltiplas variáveis
    x, y, z := 1, 2, 3
    fmt.Printf("Coordenadas: (%d, %d, %d)\n", x, y, z)
}
📝 Saída esperada: 
Sistema: TechCorp v1.0.0
Nome: João Silva
Idade: 25 anos
Altura: 1.75 m
Peso: 70.5 kg
IMC: 23.02
Área do círculo: 78.54
Anos restantes até 100: 75
Usuário com permissão de administrador
Coordenadas: (1, 2, 3)

Resumo e Boas Práticas

✅ Boas Práticas

  • Use := para declarações curtas dentro de funções
  • Use var para variáveis em nível de pacote
  • Prefira tipos explícitos quando a clareza é importante
  • Use const para valores que não mudam
  • Aproveite o iota para criar enumerações
  • Sempre verifique erros ao converter strings para números
  • Cuidado com overflow em conversões numéricas
  • Use float64 como padrão para decimais
  • Use int como padrão para inteiros (a menos que precise de tamanho específico)
  • Nomes de variáveis devem ser descritivos e em camelCase

⚠️ Erros Comuns

  • Tentar usar := fora de funções
  • Declarar variável e não usar (Go não compila)
  • Esquecer de converter tipos explicitamente
  • Ignorar erros em conversões de string
  • Usar tipos numéricos muito pequenos causando overflow
  • Confundir = (atribuição) com := (declaração)
  • Tentar modificar constantes

Exercícios Práticos

🎯 Desafios

1. Calculadora de Conversão de Temperatura

Crie um programa que converta temperaturas entre Celsius, Fahrenheit e Kelvin. Use constantes para as fórmulas de conversão.

2. Sistema de Permissões

Crie um sistema de permissões usando iota para definir níveis (Guest, User, Moderator, Admin). Atribua uma permissão a um usuário e verifique se ele pode executar determinadas ações.

3. Calculadora de Tamanhos de Arquivo

Crie constantes para KB, MB, GB e TB usando iota. Faça um programa que converta bytes para a unidade mais apropriada.

4. Validador de Tipos

Crie um programa que receba uma string do usuário e tente convertê-la para int, float64 e bool, reportando quais conversões foram bem-sucedidas.

5. Calculadora de Idade

Dado um ano de nascimento (int), calcule a idade atual, quantos dias a pessoa viveu (aproximadamente), e mostre se é maior de idade (use const para definir a maioridade).
